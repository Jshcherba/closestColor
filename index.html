Hi there!

Using this pseudo app you can get the closest color(s) for one you need.

Click here if you want to know how it works

In the document below you can find a form with 3 elements

  • Input
  • Textarea
  • Button

Please paste color VALUE (like #69849b) into the input
Please paste color variables from your _variables.scss file into the textarea

It should look like
$brand-primary: #69849b;
$brand-primary-bright: #6286a3;
$brand-primary-dark: #5f7b92;
$black: #000;
$dark-gray: #2f333a;
$new: #838588;
$medium-gray: #67696d;
$light-gray: #d5d5d5;
$white: #fff;
$lighter-gray: #f2f2f2;
$accent-first: #87788a;
$accent-second: #7d6585;
$accent-third: #cdbf88;
$accent-forth: #568f8f;
$accent-fifth: #9b373e;
$accent-six: #5a6b8a;
$accent-seven: #3d8296;
$accent-eight: #799a7f;
$dark-beige: #ae9d5c;
$slate-grey-two: #656a6e;
And then submit the button by clicking the submit button
